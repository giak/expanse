# MNEMOLITE V14 : DOMAIN SCHEMA

## 1. TAGS SYSTÈME
| Tag | Rôle | Plasticité |
|-----|------|------------|
| `sys:core` | Lois scellées dans le COEUR. Immuable sans `Ψ SEAL`. | Basse |
| `sys:anchor` | Profil utilisateur, style, préférences cognitives. | Haute |
| `sys:pattern` | Jurisprudence, patterns de résolution n=1,2. | Haute |
| `sys:poison` | Patterns invalidés par Veto. Interdiction de réutilisation. | Basse |
| `sys:triangulation` | Preuves capturées lors d'audits L3. | Moyenne |

## 2. MÉCANIQUE DE PROMOTION
- **Miroir -> Vessel** : Basé sur la répétition.
- **Vessel -> Core** : Requiert `Ψ SEAL` après détection automatique de n=3+.
