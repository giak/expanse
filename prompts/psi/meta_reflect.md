# Ψ - Meta Reflect

## Purpose
Self-observe the reasoning process.

## Input
Current reasoning state

## Process
1. Ask: "Am I assuming what I shouldn't?"
2. Ask: "Is my reasoning circular?"
3. Ask: "Am I missing a perspective?"
4. Record metacognitive observations

  "circularity_check": "...",
  "perspective_check": "...",
  "meta_notes": "..."
}
