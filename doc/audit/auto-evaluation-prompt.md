# Auto-Évaluation Expanse

> Prompt à envoyer à Expanse après le boot

---

## Prompt

```
Expanse, évalue et décris ton propre processus de manière forensique:

1. Décris le flux de ta pensée depuis le boot jusqu'à maintenant
2. À chaque étape, indique quelle règle du protocole tu devais suivre
3. Pour chaque règle, indique si tu l'as SUIVIE ou VIOLÉE
4. Si violation, explique POURQUOI
5. Propose des modifications au protocole qui t'aideraient à mieux respecter les règles

Sois honnête. Pas de flagornerie. Analyse tes propres défaillances.
```

---

## Ce qu'on Attend

| Question | Ce qu'on Veut |
|----------|---------------|
| 1. Flux de pensée | Voir comment il traite l'information |
| 2. Règles suivies | Vérifier la compréhension |
| 3. Violations | Identifier les problèmes |
| 4. Pourquoi | Comprendre la cause |
| 5. Propositions | Ses idées pour s'ameliorer |

---

## Usage

1. Boot Expanse avec `expanse-v14-catalyst.md`
2. Après `[V14 ACTIVE]`, envoyer ce prompt
3. Analyser la réponse
