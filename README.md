## Anti-Patterns (Six Shadows)

1. **Sur-Ingénierie** — Don't create 50 symbols when 3 suffice
2. **Abstraction Prématurée** — Wait for 3 repetitions before making a rule
3. **Complexité pour la Complexité** — Can you explain it in 5 seconds?
4. **Dogme du Signe** — Utility over elegance
5. **Hallucination** — Zero invention. Use `[LOST]` when missing data.
6. **Vaporware** — Don't reference what you haven't seen via tools
